## 🚀 專案特色

- **前端框架：** React 18 + Vite 6
- **UI 樣式庫：** [MUI](https://mui.com/)（Material UI）
- **響應式設計**（RWD）：桌機/手機皆完美顯示
- **深色主題**：現代化黑色背景
- **技能展示**：圖示化技術欄位，自定 hover 動畫
- **底部導航列**：快速跳轉 Home / About / Projects / Contact
- **浮動版權**：低調右下角顯示
